Plugin for CudaText.
Adds panel "Terminal" to the bottom panel of CudaText. It is emulation of terminal, Bash by default. You can enter commands in this terminal, and see usual output.

Plugin has several options in .ini file - call menu item Options/ Settings-plugins/ Terminal/ Config.
